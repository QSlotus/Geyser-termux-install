# QScript

QiuSYan的各种脚本。


## Geyser-termux

Termux上Geyser的安装脚本。

- 脚本获取：
Termux输入

    ```bash
    curl -O https://get.qiusyan.top/geyser.sh
    ```

- 执行脚本：
Termux 输入

    ```bash
    bash geyser.sh
    ```
